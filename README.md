# 🚀 Precision Lab - Hardware Intelligence

**Precision Lab (PcWeb)** là một dự án ứng dụng web hiện đại chuyên sâu về đánh giá và so sánh linh kiện phần cứng máy tính (PC Hardware). Website được xây dựng với mục tiêu cung cấp trải nghiệm Hi-tech, chuyên nghiệp và tối ưu cho những người đam mê công nghệ.

---

## ✨ Tính năng nổi bật

- **🎨 High-End Design System:** Giao diện hiện đại sử dụng ngôn ngữ thiết kế **Glassmorphism**, phông chữ **Space Grotesk** & **Inter**, mang lại cảm giác cao cấp và công nghệ.
- **🌓 Dark Mode:** Hỗ trợ chuyển đổi giao diện Sáng/Tối linh hoạt giúp bảo vệ mắt và tăng tính thẩm mỹ.
- **🌐 Đa ngôn ngữ (i18n):** Hỗ trợ đầy đủ hai ngôn ngữ Tiếng Anh và Tiếng Việt, cho phép người dùng chuyển đổi tức thì.
- **🔍 Hệ thống lọc & Tìm kiếm:** Bộ lọc linh kiện thông minh theo danh mục (CPU, GPU, RAM,...) kết hợp tìm kiếm theo từ khóa thời gian thực.
- **⚖️ Comparison Engine:** Công cụ so sánh thông số kỹ thuật (Specs) giữa hai linh kiện đặt cạnh nhau, tính toán chênh lệch hiệu năng (%) trực quan.
- **📱 Responsive Design:** Tối ưu hóa hiển thị hoàn hảo trên mọi thiết bị từ Mobile, Tablet đến Desktop.
- **🛠️ Admin Dashboard:** Khu vực quản trị chuyên nghiệp với hệ thống bảo mật bằng `LocalStorage`, cho phép quản lý nội dung dự án.
- **🛒 Dynamic Content:** Dữ liệu được quản lý tập trung và render động qua JavaScript, giúp dễ dàng cập nhật và bảo trì.

---

## 🛠️ Công nghệ sử dụng

- **Cốt lõi:** HTML5 (Semantic), CSS3 (Modern features like Flexbox, Grid, Backdrop-filter).
- **Logic:** Vanilla JavaScript (ES6+), LocalStorage API.
- **Framework & Thư viện:**
    - **Tailwind CSS:** Tối ưu hóa giao diện và tốc độ thiết kế.
    - **Swiper.js:** Xây dựng hệ thống Slider/Carousel mượt mà cho Banner.
    - **Google Fonts & Material Icons:** Hệ thống font chữ và icon cao cấp.

## 📂 Cấu trúc dự án

```text
PcWeb/

├── assets/              # Tài nguyên tĩnh
│   ├── css/             # Stylesheets (style.css, custom components)
│   ├── js/              # Logic xử lý (main.js, auth.js, i18n.js, db.js)
│   └── images/          # Hình ảnh linh kiện và media
├── pages/               # Các trang chức năng
│   ├── products.html    # Danh mục sản phẩm & Lọc

│   ├── registration.html # Trang đăng ký thành viên
│   └── admin-dashboard.html # Trang quản trị

```

---

## 🚀 Hướng dẫn khởi chạy

1. Tải toàn bộ source code về máy.
2. Mở file `index.html` bằng trình duyệt web hoặc sử dụng tiện ích **Live Server** trên VS Code để có trải nghiệm tốt nhất.
3. Chào mừng bạn đến với thế giới của **Precision Lab**!
